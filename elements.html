List of all HTML5 elements

Definition

HTML is a markup language for the structure and the presentation of a web pages . It is stand for Hypertext Markup Language. HTML shows structural semantics such as headings, paragraphs, links, and other items.

Elements in HTML5

  • <a>:it represents an abbreviation or acronym.
  • <abbr>:it defines abbreviations.
  • <article>:it can be used to contain posts.
  • <aside>: use to make sidebars that are placed next to the content.
  • <audio>: use to place an audio.
  • <b>: is for bolding text.
  • <blockquote>: is for quoting from an external source.
  • <body>: establishes the body of an HTML document.
  • <br>: use when we want a line break.
  • <button>: specifies a buttoh that users can click.
  • <canvas>:use for rendering graphics on the fly, typically in conjunction with a scripting language.
  • <caption>: use with table. Put it after the table element.
  • <cite>: use for citing.
  • <code>: use for code examples
  • <col>:represents one or more columns in a columns group.
  • <colgroup>: is a group of columns.
  • <datalist>:Specifies predefined data for an input element.
  • <dd>: it is for description lists.
  • <del>: is a Mark text as deleted without actually deleting it.
  • <dfn>:use when first defining a term.
  • <dialog>:represents a part of an application that the user interacts with to perform a task.
  • <div>: is a kind of generic container.
  • <dl>: use for describing lists.
  • <dt>: is for describing lists.
  • <em>:use to emphasis words.
  • <fieldset>: is a group related items in a form.
  • <footer>:is a footer of a document, below the main content.
  • <form>: gives forms for user input.
  • <h1 to h6>: represents a section heading.
  • <head>: represents a collection of metadata for documents.
  • <header>:represents introductory content for its nearest ancestors sectioning content.
  • <hr>:this can be used for separating text within a paragraph.
  • <html>: is the root level tag of an HTML document.
  • <i>:use for italic words.
  • <iframe>: represents a nesting browsing context.
  • <img>: use for displaying images on a webpage
  • <input>: is used with various attributes to create form input elements.
  • <ins>: is typically rendered as underlined text.
  • <kbd>:is for keyboard input.
  • <label>: is for assigning labels to interface elements such as forms.
  • <legend>: is useful for making forms easier to understand.
  • <li>: represents a list item with an ordered list or unordered list.
  • <link>:is used for linking to external resources.
  • <main>:is used for enclosing the main content of an HTML document.
  • <mark>: is essentially used for highlighting.
  • <meta>: is used for specifying various meta information about the document.
  • <nav>: is used for specifying a navigational region within a document.
  • <ol>: use with list item.
  • <optgroup>: use for grouping objects.
  • <option>: use within the select tag to specify an option.
  • <p>: represents a paragraph.
  • <param>:defines parameters for plugin invoked by object elements.
  • <progress>:displays the progress of an action, such as with a script.
  • <q>:represents a quote from some source.
  • <ruby>:is for Ruby annotation.
  • <samp>:represents sample or quoted from another program system.
  • <script>: allows authors to include dynamic script and data block in their documents.
  • <section>:is a nonspecific means of breaking up content within a webpage into sections.
  • <select>:uses with option element for selecting a particular option.
  • <small>:is useful for comments within the main content.
  • <source>:is used with audio and video elements for specifying media source.
  • <span>: is useful for applying styles to or around text, especially inline.
  • <strong>:represents strong importance, seriousness, or urgency for its contents
  • <style>:is for declaring style sheets within a document.
  • <sub>:is for displaying subscript.
  • <sup>:is for displaying superscript.
  • <table>: is for making tables in an HTML page.
  • <tbody>: represents table rows.
  • <td>: represents a data cell within a table.
  • <textaria>: is for multi-line text entry.
  • <tfoot>: is used for a table footer.
  • <th>: is for a table cell.
  • <thead>: represents the blocks of rows that consist of the column labels for the parents table elements.
  • <time>:specifies the date/time within a document.
  • <title>:represents the title of an HTML document.
  • <tr>: is a row within a table.
  • <track>: specifies text tracks with audio and video elements.
  • <u>:is useful for annotating text for various reasons.
  • <ul>:is an unordered list. Use with list item to make unordered lists.
  • <var>:defines a variable within the content of an HTML page.
  • <video>:is for embedding video within an HTML page
  • <wbr>:is an optional line wrap.

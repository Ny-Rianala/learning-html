List of HTML rules

  • You must define a doctype at the top.
  • Use lowercase for the element name.
  • indent very well the child element.
  • Tags are always surrounded by angle brackets: less-than and greater-than characters.
  • Do not use a label to wrap a group of element.
  • attribute value must have quotes.
  • attributes are separated by spaces.
  • It needs to be lowercase.
  • Element types have private attributes.
  • No double quotes inside of double quotes.

Empty elements

It is good to know the elements that don't need end tags, they are known as empty tags or elements. Have a look at the list below.

  1. <br>
  2. <hr>
  3. <img>
  4. <input>
  5. <link>
  6. <meta>

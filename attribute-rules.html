Attribute rules

Attribute value

In this rule, attribute extends a tag and also it changes its behaviour. Here, it is about the way how to put the attribute. Besides, don't forget to use quotes because it must have quotes. However, don't put them inside another.

Don't do this:

eg:

<input type=text name=search>
Do this instead:

This is the only way that we have to follow, have a look at the example below:

<input type="text" name="search">
